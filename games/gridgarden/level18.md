# Level 18 : order

Introducing the `order` property in order to **override item order** with an integer.

Default value is 0 for all items.

# Exercise

![level 18](./level18.png)

Code for water : 

```css
#water {
  order: 0;
}
```

# Solution

:bulb: Basically just apply : 

```css
#poison {
  order: 1; /* as weed is the last item of the list */
}
```
