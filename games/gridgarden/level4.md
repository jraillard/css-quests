# Level 4 : grid-column-start & grid-column-end

Combining `grid-column-start` & `grid-column-end` practicing.

:bulb: **grid-column-end** doesn't necessarily have to be bigger than **grid-column-start**, we can reverse it.

# Solution

  grid-column-end: 2;
}
